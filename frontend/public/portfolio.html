Funcionalidade Exclusiva

Sistema RPA Avançado

O TVDEFleet inclui um sistema de automação RPA (Robotic Process Automation) que permite extrair dados automaticamente de plataformas que não disponibilizam APIs.

  • RPA Designer (Admin)

    Grave scripts no seu computador e faça upload para a plataforma

  • Execução Automática

    Parceiros executam com as suas credenciais, sem ver o código

  • Plataformas Suportadas

    Uber, Bolt, Via Verde, Prio, + qualquer outra via Designer

rpa_script.py 
async def executar(page, credenciais):
    email = credenciais.get("email")
    password = credenciais.get("password")
    
    # Navegar para o site
    await page.goto("https://viaverde.pt")
    
    # Fazer login
    await page.fill("#email", email)
    await page.fill("#password", password)
    await page.click("button[type='submit']")
    
    # Extrair dados
    dados = await extrair_movimentos(page)
    
    return {"sucesso": True, "dados": dados}
